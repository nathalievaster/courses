Moment 1 - Frontend-baserad webbutveckling

I denna uppgift ska vi skapa en webbplats från grunden och använda Node.js tillsammans med npm för att automatisera olika utvecklingsuppgifter som minifiering, transpilering och optimering av filer. Webbplatsen ska bestå av HTML, CSS och JavaScript, och innehålla minst tre sidor: en startsida, en bildsida med fotografier, samt en sida där vi beskriver arbetsprocessen för utvecklingen och publiceringen av webbplatsen.

För att effektivisera arbetet och underlätta hanteringen av koden, ska vi använda npm och verktyg som Parcel, Gulp eller liknande för att automatisera uppgifter som att minifiera CSS och JavaScript, transpilera JavaScript för att säkerställa bakåtkompatibilitet och optimera bilder genom att konvertera dem till moderna format som WebP eller AVIF.

Målet med uppgiften är att bygga en webbplats med en bra filstruktur och arbetsflöde, och därefter publicera den automatiskt på en webbhost genom att använda Git och GitHub för versionshantering och publicering. Vi ska skapa en så kallad boilerplate, en förkonfigurerad projektstruktur, som kan användas som grund för framtida projekt och som underlättar utvecklingsarbetet för både nuvarande och kommande webbutvecklingsprojekt.
